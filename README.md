# IP-Check
Easy bash script to check wether the ip is available or not

## Getting Started

 ```shell
  chmod +x ip.sh
``` 
 
```shell
  ./ip.sh
```

It will generate final_ip list as well as logs for the bash script file.

## If you want to count the no of IP that are in list 

```shell
  >>> python3 ipcount.py
```
